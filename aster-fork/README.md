# Sentinel Rebrand & OpenClaw Integration

This directory contains the necessary assets, configuration (`brand.json`), and runbook (`SKILL.md`) to rebrand the `aster-mcp` project into **BitOn.Pro Sentinel**.

**Crucial integration:** This framework specifically hooks Sentinel into the `OpenClaw` ecosystem, ensuring seamless Event Forwarding (notifications, SMS, status events) directly to Claude over WhatsApp/Telegram.

Use `bootstrap.sh` or ask an AI agent to execute the instructions outlined in `SKILL.md` to perform the complete white-labeling process.
